Este es un ejemplo de una Página Web administrable para Cafeterías, bebidas o Restaurantes.

  • La Página Web Administrable está desarrollada en Angular 17.
  • Incluye un Formulario de Consultas y un Mapa de ubicación.
  • Al ser una Página Web Administrable, incluye video tutorial para su actualización.
  • Se pueden crear Landing Pages.
  • Se puede quitar y agregar páginas.
  • El diseño es Responsive, es decir la página web se adapta a Pc, Tablet y Celulares.
  • Aun no se incluye enlaces a Redes Sociales y Botón para chat WhatsApp.
